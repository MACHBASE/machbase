## Reading data with *tql* via HTTP-GET

The simplest example of providing query result in *tql* is like below code.

```js
SQL(`select * from example limit 0, 10`)
JSON()
```

The various output formats are possible.

- `JSON()`
- `JSON(transpose(true))`
- `CSV()`
- `CHART_LINE()`
- `CHART_SCATTER()`
- `CHART_BAR()`
